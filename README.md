# Web app boilerplate

- Next.js with SSR
- Custom Express.js server
- Firebase Auth + Database
- Continuous deployment to GCP Cloud run
- Context API
- Material UI

## Run locally

- Clone the repo
- Run `yarn install`
- Rename `env.example` to `.env` and add your Firebase project credentials
- Run `yarn run dev`
